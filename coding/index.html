AI Coding 工具

AI Coding 工具集

这里收集了 AI 辅助编程的常用工具，希望能帮助你提高编码效率。

AI 代码助手

在线工具

  • GitHub Copilot - AI 代码助手，支持多种编程语言
  • OpenAI Codex - OpenAI 开发的代码生成模型
  • Amazon CodeWhisperer - AWS 推出的 AI 代码助手
  • Tabnine - 基于 AI 的代码补全工具
  • Codeium - 免费的 AI 代码助手

本地工具

  • Cursor - 基于 VS Code 的 AI 代码编辑器
  • GitHub Copilot CLI - 命令行 AI 助手

代码生成

  • Replit Ghostwriter - Replit 的 AI 代码生成工具
  • CodePal - AI 代码生成和重构工具
  • Snyk Code - AI 驱动的代码安全分析
  • SourceAI - AI 代码生成和解释工具

代码审查

  • DeepCode - AI 代码审查工具
  • CodeGuru - AWS 代码审查和性能分析工具
  • SonarQube - 代码质量和安全分析平台
  • CodeClimate - 代码质量分析工具

代码解释

  • ExplainCode - 代码解释工具
  • Code2flow - 代码可视化工具
  • CodeBERT - 代码理解模型
  • Docstring Generator - 自动生成文档字符串

代码重构

  • Refactoring Guru - 代码重构指南和工具
  • Code Refactor - AI 代码重构工具
  • Sourcery - AI 代码重构工具
  • PyCharm Refactoring - PyCharm 的代码重构功能

代码搜索

  • Kite - AI 代码搜索和补全工具
  • Sourcegraph - 代码搜索和导航工具
  • GitHub Code Search - GitHub 代码搜索
  • CodeSearchNet - 代码搜索数据集

编程学习

  • Codecademy - 交互式编程学习平台
  • LeetCode - 编程练习平台
  • HackerRank - 编程技能评估平台
  • FreeCodeCamp - 免费编程学习平台
  • Coursera - 在线课程平台

开发环境

IDE 插件

  • GitHub Copilot for VS Code - VS Code 的 Copilot 插件
  • AI Assistant for PyCharm - PyCharm 的 AI 助手插件
  • Codeium for IntelliJ - IntelliJ 的 Codeium 插件

云开发环境

  • GitHub Codespaces - 云开发环境
  • GitPod - 在线 IDE
  • Replit - 在线编程平台
  • CodeSandbox - 在线代码编辑器

代码部署

  • Vercel - 前端部署平台
  • Netlify - 静态网站部署平台
  • Heroku - 云平台即服务
  • AWS Lambda - 无服务器计算服务
  • Google Cloud Functions - 无服务器函数

API 工具

  • Postman - API 测试工具
  • Insomnia - API 客户端
  • Swagger - API 文档工具
  • GraphQL Playground - GraphQL 测试工具

版本控制

  • GitHub - 代码托管平台
  • GitLab - 代码托管和 CI/CD 平台
  • Bitbucket - 代码托管平台
  • Azure DevOps - 开发工具集成平台

监控和调试

  • New Relic - 应用性能监控
  • Datadog - 监控和分析平台
  • Sentry - 错误监控平台
  • LogRocket - 前端监控和会话回放

其他工具

  • Stack Overflow - 编程问答社区
  • Dev.to - 开发者社区
  • Medium - 技术文章平台
  • HashiCorp Terraform - 基础设施即代码工具
  • Docker - 容器化平台
