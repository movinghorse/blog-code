# 前端研发工具集

这里收集了前端研发过程中常用的工具，希望能帮助你提高开发效率。

## 在线工具

### 代码格式化
- [Prettier](https://prettier.io/) - 代码格式化工具
- [ESLint](https://eslint.org/) - 代码质量检查工具
- [Beautify](https://beautifier.io/) - 代码美化工具

### JSON 工具
- [JSON Formatter](https://jsonformatter.curiousconcept.com/) - JSON 格式化和验证
- [JSON to CSV](https://www.convertcsv.com/json-to-csv.htm) - JSON 转 CSV
- [JSON Viewer](http://jsonviewer.stack.hu/) - JSON 可视化查看器

### 正则表达式
- [RegExr](https://regexr.com/) - 正则表达式测试工具
- [RegEx101](https://regex101.com/) - 正则表达式在线测试
- [Regular Expressions 101](https://www.regex101.com/) - 正则表达式解析器

### 颜色工具
- [Color Picker](https://www.w3schools.com/colors/colors_picker.asp) - 颜色选择器
- [Color Hex](https://www.color-hex.com/) - 颜色代码查询
- [Paletton](https://paletton.com/) - 调色板生成工具

### 图片工具
- [TinyPNG](https://tinypng.com/) - 图片压缩工具
- [Image Optimizer](https://imageoptimizer.com/) - 图片优化工具
- [SVG Editor](https://svg-edit.github.io/svgedit/releases/svg-edit-2.8.1/svg-editor.html) - 在线 SVG 编辑器

### 字体工具
- [Google Fonts](https://fonts.google.com/) - 免费字体库
- [Font Awesome](https://fontawesome.com/) - 图标字体库
- [Icon Finder](https://www.iconfinder.com/) - 图标搜索工具

- [GitHub Desktop](https://desktop.github.com/) - GitHub 桌面客户端
